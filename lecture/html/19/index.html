Formulare

ElementBeschreibung
formFormular, welches aus Labels und Eingaben besteht
labelBeschriftungen
inputverschiedene Arten von Benutzereingaben; Art über "type" Attribut spezifiziert
z. B. text, password, date, file, color, radio, checkbox, number, range
select + optionAuswahlliste
textareamehrzeiliges Eingabefeld
buttonBenutzerinteraktion durch Anklicken; "type" Attribut für Absenden (submit), oder Zurücksetzen (reset)
  • "name" Attribut um Werte den Eingabeelementen zuordnen zu können
  • "value" Attribut für vorbelegte Werte
  • Daten an Server senden und dort verarbeiten -> wird später behandelt
